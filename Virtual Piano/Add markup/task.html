Description

Of all people, developers should not have any problem with installing programs on their computers because they have to do it so often. We suggest you strengthen this skill. If you do not have node.js on your device, please install it. You can download it from the official website. This program will allow us to check that you are solving the project correctly.

Without node.js installed on your computer, you cannot complete the project.

To go through the first step, change the page title.

The <kbd> tag is used to denote the text to be typed on the keyboard or for a key name (you can read more about it on MDN). Use this paired tag in the body of the document to specify seven keys on the keyboard: clicking these will eventually create charming sounds of your instrument. You must have the symbols A, S, D, F, G, H and J placed in this tag. These will be the white keys on the piano.

If you open this HTML file in your browser, you'll see the names of the keyboard keys. These keys do not make sounds yet, but you're one step closer to creating a working virtual piano.

Objectives

  1. Change the page title.
  2. Add 7 <kbd> tags to the document body. In each of them enclose one of the symbols: A, S, D, F, G, H or J.
