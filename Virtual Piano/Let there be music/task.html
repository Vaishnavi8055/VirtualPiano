Description

Having keys that do not produce sound is certainly good, but it does not fully reflect the concept of musical instruments. Fortunately, this problem is easy to fix.

Change the event handler so that when you press the A key, an audio object is created and played. You can use the audio recordings we offer or specify yours. Check the result in a browser. Your piano has hopefully made its first sound!

If you have problems playing audio recordings, try to specify the path to them relative to the HTML file.

Perform this operation for the remaining six keys, and the stage will be considered complete.

Objectives

  1. Modify the event handlers so that when you press the keys instead of displaying messages in the console, audio objects are created and played.
